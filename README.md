# myconspect[READ ME.txt](https://github.com/user-attachments/files/24731008/READ.ME.txt)
Инструкция по пользованию "Мои конспекты"
Необходимо начать, для кого это приложение:
Оно необходимо для учащихся в школах, учителям и просто тем кто хочет освежить свои знания.



1. При запуске:
когда программа загрузилась ,на начальном экране появляются 6 иконок ,со школьными предметами
Выбрав одну из них, вам покажутся конспекты ,по нужному вам предмету. Снизу них есть кнопка"начать использование" она переносит в раздел "все конспекты"
2. Меню слева:
Начнем сверху вниз.
"Поиск"название говорит само за себя.В нем можно писать темы которые вас интересуют ,или же писать слова из конспектов, чтобы найти более точный конспект .
"Все конспекты" в этом отделе находятся  все конспекты,по всем предметам. Для удобства ,можно выбрать конкретный предмет
2.1Мои конспекты:
"Мои записи " там сохраняются все созданные , или сохранённые конспекты пользователя
"Создать конспект" при нажатии всплывает окно с созданием конспектов:
Сначала нужно выбрать предмет ,по которому будет конспект ,его тему ,написать ,или загрузить сам конспект.
Если необходимо,то есть возможность добавить шаблон,или фотографию.Он будет сохранен"мои записи"
"Импорт из файла" сразу можно загрузить готовый текстовый файл(!!! Название конспекта ,будет взято из файла!!!)он будет сохранен,как "мои записи"
2.3 Статистика:
Она находится еще ниже,в ней показываются общее число конспектов, собственные конспекты и количество конспектов по предметам.
Кнопка "обновить" обновляет информацию в статистике,или же она обновится сама,при выходе из программы
3.Меню сверху:
3.1 файл:
"Новый конспект" выполняет,те же функции ,что и  "создать конспект"
"импорт"выполняет,те же функции ,что и  "Импорт из файла"
"Экспорт все" можно сохранить вами созданные конспект, на ваше устройство
3.2Правка: выполняет,те же функции ,что и  "Поиск"
3.3 Вид:выполняет,те же функции ,что и  "Обновить"
3.4 Справка:
"О программе" в нем есть актуальная информация о версии программы ,что нового и о самой программе
"Помощь"  в этом разделе есть информация о комбинации клавиш и их последствиях
4. Работа с конспектами:
4.1 "Мои записи " при просмотре каталога конспектов ,есть две кнопки:"просмотр" и "сохранить копию".
При нажатии на первую показывается сам конспект ,есть возможность сохранить его или распечатать .
А вторая сохраняет данный конспект в "Мои записи"
4.2 "Мои записи"  имеется такой же каталог,но добавляется помимо просмотра(функции те же ,что и выше ),возможность редактировать и удалить полностью конспект
